# Diligent Design System — Video Production Defaults

## Color Palette

| Token | Hex | Usage |
|-------|-----|-------|
| White | `#ffffff` | Dominant background |
| Dark Navy | `#1a1d2e` | Primary text, icons, dark UI elements, data card backgrounds |
| Dark Red | `#e84c3d` | Sparingly — key highlights, CTAs, active states, accent lines |
| Light Gray | `#f9fafb` | Card backgrounds, sections, sidebar |
| Border Gray | `#e5e7eb` | Dividers, borders (1px standard, 2–3px emphasis in red) |
| Medium Gray | `#6b7280` | Secondary text, labels, captions |

## Usage Rules

### Backgrounds
- Main canvas: White (`#ffffff`)
- Cards/panels: Light gray (`#f9fafb`)
- Data cards (Scene 4 metrics): Dark navy (`#1a1d2e`) with white text

### Text
- Primary headings: Dark navy (`#1a1d2e`)
- Body text: Dark navy (`#1a1d2e`)
- Secondary text: Medium gray (`#6b7280`)
- On dark backgrounds: White (`#ffffff`)

### Accents (Use Sparingly)
- Red (`#e84c3d`) ONLY for:
  - CTA buttons
  - Alert badges / notification dots
  - Accent lines on data cards
  - Active/selected states
  - Key metric highlights
- Never use red for backgrounds, large areas, or decorative elements

### Borders & Dividers
- Standard: `#e5e7eb`, 1px
- Emphasis: `#e84c3d`, 2–3px
- Dividers: `#e5e7eb`, 1–2px

## Typography

| Element | Font | Size | Weight |
|---------|------|------|--------|
| Headings | Arial, sans-serif | 18–24px | 600 |
| Body | Arial, sans-serif | 13–14px | 400 |
| Labels | Arial, sans-serif | 11–12px | 400 |
| Buttons | Arial, sans-serif | 14px | 600 |
| On-screen data cards | Arial, sans-serif | 24–36px | 600 |

## Spacing & Layout
- Border radius: 6–12px for cards and panels
- Border radius: 4px for small elements (buttons, badges)
- Card padding: 15–20px
- Grid system: 8px base unit

## Video-Specific Principles

1. **White space dominates** — generous padding, no visual clutter
2. **Dark elements provide contrast** — dark navy for structure and data cards
3. **Red is precious** — maximum 2–3 red elements on screen at any time
4. **Motion is purposeful** — smooth transitions, never frantic or flashy
5. **Professional aesthetic** — corporate-ready, boardroom-appropriate
6. **Data cards animate with weight** — they should feel substantial, hold for 2–3 beats
7. **Split-screen juxtaposition** — builder tools (dark aesthetic) on left, product output (light aesthetic) on right
